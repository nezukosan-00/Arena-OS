# ArenaOS

ArenaOS is an AI-powered arena operations and fan experience platform designed for large-scale events such as the FIFA World Cup 2026. It combines intelligent crowd management, personalized navigation, multilingual assistance, accessibility support, transportation guidance, and real-time operational intelligence in a single experience.

## Development

- Install dependencies with `npm install`
- Start the app with `npm run dev`
- Build for production with `npm run build`
